Production Vision Inspection

검사 입력 안정화부터 결과 제공까지 연결한 영상처리 포트폴리오

프로젝트 개요

제조 검사 이미지는 조명, 배경, 카메라 상태에 따라 입력 편차가 발생할 수 있습니다. 그래서 이 프로젝트는 단순 OK/NG 분류보다 먼저, 검사 대상 영역을 안정적으로 추출하고 결과를 해석 가능한 형태로 제공하는 데 초점을 맞췄습니다.

주요 구현 내용

  • 원본 이미지에서 배경을 추정해 ROI를 추출하는 전처리 흐름 구성
  • ROI-only와 CLAHE 적용 버전을 비교해 입력 안정성 차이 검토
  • 이상 점수, heatmap, overlay를 함께 제공해 결과 확인성 강화
  • 운영 임계값을 조정하며 FP/FN 사례를 정리하고 최종 운영 포인트 도출
  • FastAPI 응답과 MFC 데모 클라이언트 연결로 활용 흐름까지 정리

포트폴리오에서 강조한 부분

  • 입력 전처리와 결과 해석을 함께 본 점
  • 단순 모델 소개가 아니라 운영 관점의 판단 기준을 다룬 점
  • 화면 연동과 응답 구조까지 포함해 실사용 흐름을 보여준 점

사용 기술

Python, OpenCV, NumPy, Pandas, FastAPI, MFC/C++, 이상탐지 기반 heatmap/overlay 분석
